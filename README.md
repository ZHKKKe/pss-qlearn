# pss-qlearn

Requirements: 
  - numpy, sklearn, matplotlib, pygame, pickle

To run qss-learn: 
  - cd ./src
  - python run.py

Folders:
 - docs: support files
 - result: some output images
 - src: source code
   - third_party: code from https://github.com/chncyhn/flappybird-qlearning-bot.git 
   - other: the code for our pss-qlearn
